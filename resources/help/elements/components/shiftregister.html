Creating a Shift Register

Set the number of inputs to the shift register by entering a number or using the keypad. The number of bits of the (shift) amount input will be automatically set to the ceiling of the log2 of this number (e.g., 6 inputs will result in a 3-bit selector input).

The shift type dictates the type of shift (and how sign bits are handled for right shifts).

Click on OK to place the element, or Cancel to cancel its creation.
